##### Vue常见性能优化方法

1. 路由懒加载

   ```js
     const router = new VueRouter({
         routes: [
             {path: '/foo', compponent: () => import('./foo.vue')}
         ]
     })
   ```

2. keep-alive缓存页面

   ```
   <keep-alive> //include exclude max
   	<comp></comp>
   </keep-alive>
   ```

3. v-show复用DOM

   ```vue
    <div v-show="value">
        <CacheInp value="123" />
   </div>
   <div v-show="!value">
       <CacheInp value="456" />
   </div>
   ```

4. v-for避免同时使用v-if

   ```
   外面嵌套一层加v-if 使用计算属性过滤
   ```

5. 长列表性能优化

   ```
   // 如果是纯粹的数据展示，不做响应式
   {
       data: () => {
           list: []
       },
       async created(){
           const list = await axios.get('/api/list')
           this.list = Object.freeze(list)
       }
   }
   
   // 如果是大数据长列表，可采用虚拟滚动 vue-virtual-scroller、vue-virtual-scroll-list
   ```

6. 事件的销毁

   ```
   created(){
       this.timer = setinterval(fn, 2000)
   },
   beforeDestroy(){
       clearInterval(this.timer)
   }
   ```
